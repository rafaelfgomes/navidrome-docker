# Navidrome Music Server

Navidrome is a service for manage your music collection and play it over clients. It's like having your "own Spotify".

For more information Navidrome click [here][Navidrome Site].

## How to install and run

- First make a copy of the ".env.example" file and rename it to ".env";
- Fill the variables in the .env file;

After that you can run the command:

`docker-compose up -d`

[Navidrome Site]: https://www.navidrome.org/docs/overview
